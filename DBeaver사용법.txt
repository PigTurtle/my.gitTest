DBeaver의 Database Connection 선택

MySQL(DB종류) 선택

General 탭에 정보 입력
서버호스트를 localhost로 해야함 <- 원격접속은 ssh를 통해 할 것이고, 
ssh 접속 후 로컬에서 접속하듯 연결할 것이기 때문(3306포트를 통해 직접 접속 가능하지만 보안상 비추천)
User name은 MySQLDB에서 만든 접속할 DB의 사용자 계정
finish 클릭
-> 오류 뜨는게 정상

Database Navigator에서 방금 만든 연결 아이템 마우스 우클릭 - Edit Connection 클릭
Connection Settings - Network 선택
SSH Tunnel 탭에서 Use SSH Tunnel 체크
Host/IP에 원격지 DB 서버의 주소(도메인 또은 IP주소) 입력
여기의 User Name은 DB가 아니라 서버 자체의 SSH 로그인 계정
test tunnel configuration 클릭하여 ssh접속 테스트 -> connected 뜨면 정상 실패면 SSH 정보가 잘못
test connection 클릭하여 DB접속 테스트 -> connected 뜨면 정상 실패면 처음 입력한 DB 접속 정보가 잘못

설정이 끝나면 Database Navigator에서 새로 설정한 DB를 더블클릭하여 접속하면 성공적으로 접속 확인 가능
